  • 微信小程序

    • 该项目使用 微信开发者工具，实现了电影的影评展示、音乐播放、句子欣赏功能。

      • 技术难点：

  • react 框架造轮子

    • 该项目使用 react 、Webpack和 TypeScript 搭建的一个造轮子项目。
    • 使用的技术栈：React、TypeScript、Webpack。

      • 技术难点：
      • ypeScript 的兼容性问题： "skipLibCheck": true
      • 在页面中展示代码：可以使用 require 引用 url，但是又有一个问题，require 不能展示动态地址。所以现在的解决方案是在每个需要的地方 手动添加 url =》const x= require(`!!raw-loader!${props.path}`)
      • 引入 png 图片： 还需要安装关于图片的 loader，但是在项目中还是有些问题，可以使用 @ts-ignore 来忽略问题。
      • 当在 div 中出现内联样式，会出现一部分不被遮拦的情况：用 Fragment 标签将内容包括起来，Fragment 标签不会在网页代码中现实。使用传送门 portal，动态生成组件，最后闭包传 API。
